Competencias Transversales

Competencias Transversales:

Las competencias transversales son aquellas habilidades, conocimientos y valores que se ponen en practica de manera integral en los diferentes aspectos de la vida, tanto en el ámbito personal, ámbito laboral y ámbito social. Podemos clasificar las competencias transversales de la siguiente manera: Competencias Personales, Competencias Participativas, Competencias Metodológicas y Competencias Técnicas.

Competencias Personales:

Son aquellas competencias que están relacionadas al (saber ser) como indiviudo y a los valores de la persona. Dentro de ellas tenemos: honestidad, responsabilidad, respeto, sinceridad entre muchas otras más. Ejemplo: Un cajero de una tienda que tenga la oportunidad de agarrar dinero de la caja, pero este no lo haga por que tiene principios y es incapaz de robar. Podemos decir que esta persona es competente a nivel personal debido a que tiene buenos valores.

Competencias Participativas:

Son quellas competencias que estan relacionadas a la capacidad de relacionamiento y colaboración que tenemos con otras personas, con el fin de aportar a una situación en específico. Ejemplo: Una persona que se encuentra en una reunión de vecinos y propone ideas para resolver los diferentes problemas que han surgido en el sector donde vive.

Competencias Metodológicas:

Las Competencias Metodológicas son aquellas que estan relacionas al (saber hacer), es decir, como aplicamos nuestros conocimientos para resolver una tarea específica tratando de obtener los mejores resultados. Ejemplo: La elaboración de una plato de comida por parte de un chef, ya que esta tiene una serie de paso a seguir para poder llegar al resultado final,

Competencias Técnicas:

Son aquellas competencias que estan relacionas a los conocimientos específicos adquiridos sobre una área determinada. Ejemplo: Una persona que realizó estudios especializados en una ciencia en específico. Dicha persona posee conocimientos técnicos sobre la ciencia que estudió.

Currículum:

El currículum es la agrupación de los datos personales, experiencia académica, experiemcia laboral y otros datos importantes a considerar como lo son refencias laborales y referencias personales, de un individuo. El currículum se hace con la finalidad de organizar y resumir información, para que la misma pueda ser vista de una manera más fácil.;

Carta de Presentación:

Como su nombre la indica, es una carta que contiene información para presentarnos, en dicha carta va información complementaria a la del currículum.

Habilidad:

Es el talento que posee una persona para realizar una determinada tarea con facilidad. Ejemplo: Una persona que realiza cálculos matemáticos mentalmente de manera facil, podemos decir que esta persona posee una habilidad mental.

Vocación:

Es la inclinación o inspiración que siente una persona para realizar una actividad en específico, dicha actividad debe generar en la persona una satifascción. Ejemplo: Una persona que durante mucho tiempo se dedicara a ser Gerente de un banco, pero no se sienta satisfecha al máximo y con el pasar del tiempo descubra que lo que realmente le llena es ser artesano.

Desarrollo Web

Desarrollo Web:

El Desarrollo Web, es proceso de creación y mantenimiento de páginas web, generalmente para formar parte de un sitio web disponible en la internet. El lenguaje que se usa para su desarrollo es el HTML, combinado con otros lenguajes como lo son CCS y Javascript.

HTML:

Html Significa Hyper Text Markup Language, y es un lenguaje de programación de etiquetas, decimos que es de etiquetas porque cuando programamos con el utilizamos etiquetas como "body", "head", "table" entre otras. Con este lenguaje de programación realizamos el esqueleto principal de nuestra web, es decir realizamos la estructura principal, como ira nuestra encabezad, el cuerpo de la página y nuestro pie de página adicional si nuestro sitio web contiene tablas u otros elementos.

CCS:

Css, significa Cascada Stylesheet, que quiere decir hoja de estilo en cascada porque va aplicando los estilos en forma de cascada. Con el lenguaje de programación Css podemos darle estilo visual a nuestro sitio web, con este lenguaje podemos darle tamaños a los div creados con html, podemos asignarles un determinado tamaño a las imágenes de nuestro sitio y modificar cualquier aspecto visual de nuestro sitio web.

Javascript:

Javascript es un lenguaje de programación que usa script, con este lenguaje de programación podemos darle funcionalidad y lógica a nuestro sitio web, podemos incluir operaciones matemáticas, operaciones booleanas (Verdadero y Falso). Por ejemplo, si tenemos un elemento “table” html y dentro de él tenemos información de alumnos registrados y sus notas, y queremos saber cuál es el promedio del alumno, estas operaciones podemos realizarla con JavaScript. Adicional a esto con JavaScript podemos acceder y cambiar las propiedades y elementos de los elementos creados con Html y Css.

Scrum:

Scrum es una metodología ágil que suplanta la manera antigua de desarrollar software como lo es la metodología en cascada. Scrum se enfoca en tratar de conseguir un mejor rendimiento en el menor tiempo posible, e ir realizando sprint y evaluaciones ciclícas de lo avanzado, para cumplir con los objetivos pautados del proyecto de desarrollo.

JSON:

Significa JavaScript Object Notation, es un formato de texto ligero para el intercambio de datos. JSON es como una especie de notación literal de objetos de JavaScript , debido a su amplia adopción como alternativa a XML, se considera un formato de lenguaje independiente.
